# Seedance 合规提示词检查清单

输出最终 prompt 前逐项核对。任一项不通过则继续改写。

## A. 版权与 IP

- [ ] 无真实品牌、商标、产品型号（酒类、汽车、手机、服装 logo 等）
- [ ] 无影视作品/动漫/游戏角色名或可直接识别的 IP
- [ ] 无明星、政客、网红等真实公众人物姓名或强特征描述
- [ ] 无「像某某明星」「某某同款」类表述
- [ ] 陈设与道具使用泛化描述（无标识酒瓶、普通商务装）

## B. 内容安全

- [ ] 无显性暴力、武器特写、血腥、虐待
- [ ] 无核爆、蘑菇云、战争轰炸、恐怖袭击等灾难具象
- [ ] 无色情、裸露、性暗示
- [ ] 无煽动性、歧视性、违法活动描写
- [ ] 饮酒场景不鼓吹拼酒、酗酒（可保留应酬压力与不适反应）

## C. 提示词形态（Seedance 可读性）

- [ ] 已移除剧本符号：△、场景X、角色名+冒号对白、旁白音、OS 等
- [ ] 已移除大段台词；情绪用表情、肢体、镜头表达
- [ ] 包含**地点/环境**描述
- [ ] 包含**视觉风格**（写实、电影感、色调、光照）
- [ ] 包含**镜头或动态**（推近、切镜、定格、跟拍）
- [ ] 包含**可见动作与道具**，而非抽象叙事
- [ ] 长度适中：单条建议 150–400 汉字；过长则删冗余对白背景

## D. API 与项目约束（筷子 / 本项目）

- [ ] 最终交付仅为可填入 `prompt` 的正文，不夹带 model/API 参数说明
- [ ] 未假设使用未接入的字段（如 negative_prompt、seed）
- [ ] 时长与内容匹配：5 秒单节拍或短蒙太奇；多场景已合并或建议拆分
- [ ] 结尾可含一句合规排除（无品牌、虚构人物、不出现 XX），但不冗长

## E. 常见版权报错后的快速修复

| 报错/现象 | 优先检查 |
|-----------|----------|
| 版权原因 | 品牌名、IP、真实人物 |
| 审核不通过 | 灾难、暴力、酗酒鼓动 |
| 生成偏离 | 叙事过多、可见细节不足 |
| 立即失败（有参考图时） | 参考图含真人脸或版权角色（与本 skill 的纯文生视频无关） |
