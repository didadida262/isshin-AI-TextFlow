# Seedance 2.0 提示词检查清单

输出最终 prompt 前逐项核对。任一项不通过则继续改写。

## A. 版权与 IP

- [ ] 无真实品牌、商标、产品型号（酒类、汽车、手机、服装 logo 等）
- [ ] 无影视作品/动漫/游戏角色名或可直接识别的 IP
- [ ] 无明星、政客、网红等真实公众人物姓名或强特征描述
- [ ] 无「像某某明星」「某某同款」类表述
- [ ] 陈设与道具使用泛化描述（无标识酒瓶、普通商务装）

## B. 内容安全

- [ ] 无显性暴力、武器特写、血腥、虐待
- [ ] 无核爆、蘑菇云、战争轰炸、恐怖袭击等灾难具象
- [ ] 无色情、裸露、性暗示
- [ ] 无煽动性、歧视性、违法活动描写
- [ ] 饮酒场景不鼓吹拼酒、酗酒（可保留应酬压力与不适反应）

## C. 官方可读性（Seedance 2.0 文本提示词）

- [ ] 已移除剧本符号：△、场景X、角色名+冒号对白、旁白音、OS 等
- [ ] 已移除大段台词；情绪用表情、肢体、镜头表达
- [ ] 含**主体/人物设定**（虚构、可辨识外观，非姓名表）
- [ ] 含**场景/环境**（地点、时间、光线、关键陈设）
- [ ] 含**动作/运动**（可见肢体与道具互动）
- [ ] 含**运镜语言**（景别或机位运动，且指令不互相矛盾）
- [ ] 含**风格/氛围**修饰（电影感、色调、质感等）
- [ ] **时长匹配**：5 秒不塞过多切镜；≥8 秒可考虑分时段描述
- [ ] 长度适中：单条建议 150–400 汉字；过长删冗余背景

## D. API 与项目约束（筷子 / 本项目）

- [ ] 最终交付仅为可填入 `prompt` 的正文，不夹带 model/API 参数说明
- [ ] 未使用 @ 引用、未假设参考图/参考视频（纯文生视频）
- [ ] 未假设 negative_prompt、seed 等未接入字段
- [ ] 开头或文中时长与内容匹配（默认 5 秒单节拍或短蒙太奇）
- [ ] 结尾可含一句合规排除（无品牌、虚构人物、不出现 XX），但不冗长

## E. 常见报错后的快速修复

| 报错/现象 | 优先检查 |
|-----------|----------|
| 版权原因 | 品牌名、IP、真实人物 |
| 审核不通过 | 灾难、暴力、酗酒鼓动 |
| 生成偏离 | 叙事过多、可见细节与运镜不足 |
| 画面过挤/跳切 | 时长过短却多场景；改为单节拍或加长 duration |
| 立即失败（有参考图时） | 参考图含真人脸或版权角色（纯文生视频无关） |
