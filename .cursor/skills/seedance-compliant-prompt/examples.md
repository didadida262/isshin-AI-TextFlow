# Seedance 合规提示词示例

## 示例 1：商务应酬剧本 → 合规单条（本项目真实案例）

### 输入（高风险片段）

- 含「茅台」品牌
- 含「蘑菇云」「倒计时」
- 剧本格式 △、角色对白、旁白

### 输出（合规合并 prompt）

```
5秒电影感竖屏短片，写实风格，暖黄低照度转冷白。中式高端会所包间夜景：深色木圆桌上有无标识白酒瓶与玻璃杯，年轻男职员举杯强笑一饮而尽，指尖微颤、额头细汗，暗中按住腹部；对面中年上司举杯似笑非笑，服务员入画倒酒至杯沿，男职员挤出职业微笑，握杯的手短暂僵住。镜头切向落地窗，远处天际骤然亮起刺眼白光，室内暖光被压暗，背景音戛然而止，三人同时愣住望向窗外，一人手中酒杯悬在半空，窗外可见远处云状光晕缓慢升腾，气氛诡谲压抑。人物均为虚构面孔，无字幕、无品牌标识，不出现爆炸、蘑菇云、武器、倒计时或核灾难画面。结尾定格在人物惊愕侧脸与窗外异光。
```

### 已替换项

- 茅台 → 无标识白酒瓶
- 小张/王总/老板 → 年轻男职员、中年上司
- 蘑菇云 → 云状光晕 / 刺眼白光
- 倒计时旁白 → 删除
- 剧本符号与对白 → 动作与镜头

## 示例 2：动作句扩写（过滤器友好）

| 避免 | 改为 |
|------|------|
| 士兵在街上开枪 | 广角镜头，废墟街道薄雾弥漫，穿旧式军装士兵向画外方向射击，烟尘升腾，手持纪录片质感，阴天平光 |

扩写时补齐：地点、光线、机位、氛围。

## 示例 3：连接测试用短 prompt

```
5秒写实风格，一只可爱的柯基犬在开满向日葵的田野里欢快奔跑，阳光明媚，浅景深，镜头低角度跟随，无文字无 logo，虚构宠物场景。
```

说明：测试 prompt 也应避免真实品牌与版权角色；动物场景用泛化描述即可。

## 示例 4：多场景处理策略

**用户要求「合并成一条」+ duration=5**

- 用「镜头切向」「随后」串联两节拍
- 每节拍只保留 1–2 个可见动作
- 在回复中注明：若画面过挤，建议 `duration` 改为 10 或分两次生成

**用户接受拆分**

- 场景 1 一条 prompt（应酬压力）
- 场景 2 另一条 prompt（窗外异象）
- 后期在剪辑软件拼接
